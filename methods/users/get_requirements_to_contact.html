GetRequirementsToContactRequest

Both users and bots may be able to use this request. See code examples.

---functions---
users.getRequirementsToContact#d89a83a3 id:Vector<InputUser> = Vector<RequirementToContact>

Returns

A list of the following type is returned.

RequirementToContact

This type can be an instance of either:

RequirementToContactEmptyRequirementToContactPaidMessages
RequirementToContactPremium

Parameters

idInputUserAnything entity-like will work if the library can find its Input version (e.g., usernames, Peer, User or Channel objects, etc.). A list must be supplied.

Known RPC errors

This request can't cause any RPC error as far as we know.

Example

from telethon.sync import TelegramClient
from telethon import functions, types

with TelegramClient(name, api_id, api_hash) as client:
    result = client(functions.users.GetRequirementsToContactRequest(
        id=['username']
    ))
    for x in result:
        print(x)
